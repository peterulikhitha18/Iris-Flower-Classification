# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview

This project is a Machine Learning application that predicts the species of an Iris flower based on its measurements.

The model classifies Iris flowers into:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

The project uses a K-Nearest Neighbors (KNN) algorithm and is deployed using Streamlit.

---

## 🎯 Objective

The main goal of this project is to build an end-to-end Machine Learning project including:

- Data loading
- Data preprocessing
- Exploratory Data Analysis
- Model training
- Model evaluation
- Deployment
